## Getting the Code

Clone it using git:

    $ git clone git://github.com/jeffallen6767/matrix.git
    $ cd matrix


Install all dependencies:

    $ yarn install


Start the webserver:

    $ yarn start


View in a web browser:

[http://localhost:3333/](http://localhost:3333/)
